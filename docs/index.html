How it works

Three steps, then you're talking to your account.

You don't install anything yourself. You don't read docs. You ask your AI agent to set it up, paste a token once, and start asking questions.

Step 1

Connect once.

Tell Claude Code or Cowork to install linkedin-ads. It does. You paste your LinkedIn token once. The token stays on your machine, never leaves it.

Step 2

Ask anything.

"Audit my account." "Which audiences do I actually use?" "Pause everything under 0.3% CTR." Your agent translates the question, reads your account, comes back with answers.

Step 3

Act with confidence.

When the agent wants to change something, it shows you exactly what before doing it. Dry-run mode previews the impact. Nothing fires until you say yes.

Questions

What people ask before they try it.

Do I need to know how to code?
No. Your AI agent runs everything. You ask in English, the agent does the work. If you've used Claude Code or Cowork once, you already know how to use this.
Is it safe? What if the AI does something I didn't want?
Every change is previewed before it ships. The agent shows you exactly what it's about to do — old value, new value, which campaign — and waits for your confirmation. Dry-run mode lets you see the impact without anything actually happening. There's no bulk delete and no silent edits.
Does this replace Campaign Manager?
No. Campaign Manager is still where you build the heavy stuff visually. linkedin-ads is what you talk to when you want answers, audits, and quick changes — the things you'd otherwise do in spreadsheets or by clicking around for an hour.
Where does my LinkedIn token live?
On your machine, in a config file only your user account can read. It never leaves your computer. The tool talks to LinkedIn directly — there's no middle server, no third party, no telemetry.
Which LinkedIn accounts are supported?
Any ad account you have access to. Personal, business, agency-managed — if you can see it in Campaign Manager, your agent can read it here.
Is it free?
Yes. Open source under the MIT license. No paid tier, no signup, no account. The only thing you need is a LinkedIn API token, which is also free.
